# Визуализация сетевой топологии

## Стек

- Next.js 15
- TypeScript
- Cytoscape.js + cytoscape-dagre для построения графа
- Tailwind CSS

## Запуск

1. Клонировать репозиторий и перейти в папку

```bash
git clone https://github.com/D1flector/Topology-app.git
cd topology-app
```

2. Установить зависимости

```bash
npm install
```

3. Запустить проект

```bash
npm run dev
```

4. Открыть в браузере

```
http://localhost:3000
```
